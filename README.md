## Overview
This project analyzes Zomato restaurant data using Python and visualizes insights through an interactive Streamlit dashboard.  
It covers data cleaning, exploratory data analysis (EDA), and dashboard creation.

## Dataset
Due to Github's file size limit,the dataset is not included here
You can download it from the kaggle[https://www.kaggle.com/datasets/rabhar/zomato-restaurants-in-india?hl=en-IN


##  Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn, Plotly  
- Streamlit  
- Jupyter Notebook

- `Zomato.ipynb`

## Output

- Average ratings by cuisine  
- Restaurant count by city  
- Cost distribution  
- Interactive filters for city and rating
